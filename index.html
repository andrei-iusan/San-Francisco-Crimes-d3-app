San Francisco

This plot represents the number of thefts per day, averaged over 30 days. The green area represents the number of theft cases that were solved by the police.

It is clear from the plot that the number of thefts remained stable from 2003 to 2012, but then started to increase.

You can normalize the plot. This will tell you the proportion of thefts that were solved, rather than the actual number of thefts. This tells us that if the number of thefts is higher, there are more cases that are not solved by the police.
