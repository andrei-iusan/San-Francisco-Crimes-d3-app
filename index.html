San Francisco

Data about crime activity in San Francisco is publicly available starting from January 1st, 2003 at data.sfgov.org. This data comes from the San Francisco Police Department system.

I explored this data set using R. My analysis can be found here:
San Francisco Crimes Investigation

I will present a summary of my findings. You can navigate through the presentation using the arrows from the upper right corner.

This plot represents the number of crimes per day, averaged over 30 days.

You can normalize the plot. This will tell you the proportion of solved crimes rather than the actual number of crimes.

On the un-normalized plot there is an increasing trend in the number of crimes from 2010 until the end of 2015. The number of solved crimes is higher for 2013 and 2014, but it otherwise it stays at about 150 solved crimes.

On the normalized the plot we see this increase in crime activity combined with the relative stable number of solved crimes, translated in a decrease in the percentage of solved crimes, especially in 2015.

Next we will inspect a few particular crimes that caused this effect.

This is the same type of chart, but the data is filtered for the "Larceny/Theft" category. Note that this is a very common type of crime, and it became more common in the last years.

You can normalize this graphic like the previous one, and also superimpose the total number of crimes.

You can see that theft accounts for a lot of variability in the total number of crimes.

Now let's have a look at the evolution of drug related crimes. This is also a very common crime, it's the 6th most common crime. I chosed to present this category because like theft, the number of drug related crimes changed drastically in the last few years. In this case, however, the number of crimes decreased in the last years.

Please note that the percentage of solved crimes in this category is very high.

The combination of increasing theft activity and decreasing drug activity led to the result we saw in the overview of all crimes, that the percentage of solved crimes dropped from 2014 to 2015.

We can see now that the situation is not due to a change in the way police handels crimes, but due to the changing of the number of crimes in certain categories (particularly increased theft activity) that are difficult to solve.
