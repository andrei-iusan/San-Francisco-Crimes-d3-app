San Francisco

This plot represents the number of crimes per day, averaged over 30 days.

You can normalize the plot. This will tell you the proportion of solved crimes rather than the actual number of crimes.

On the un-normalized plot we see an increasing trend in the number of crimes from 2010 until the end of 2015. The number of solved crimes is higher for 2013 and 2014, but otherwise it stays at about 150 solved crimes per day.

On the normalized plot we see this increase in crime activity combined with the relative stable number of solved crimes translated in a decrease in the percentage of solved crimes, especially in 2015.
